jQuery - Google Analytics Scrolling Tracker

Does your visitor scroll to see the content?

Using this simple tracking plugin for Google Analtyics you can add tracking to your HTML-elements. If the user scrolls so that they are visible on the screen - it will be tracked in Google Analtyics using the event tracking-feature.
